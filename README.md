#login-form project
A simple **login form** built with **HTML and CSS**.
This project is part of my journey learning **frontend development** and practicing form design and user interaction.

##Features
-User friendly login form interface
-Responsive design (adopts to different screen sizes)
-Simple input validation using HTML5 attributes
-Styled entirely with CSS
